seeit3d by davidmr

An Eclipse plugin for software visualization 3D

Installation

In order to install SeeIT 3D, it is required to previously install the Eclipse IDE. It can be found at www.eclipse.org at the Downloads section. Once Eclipse is installed, SeeIT 3D can be installed. To do so, open Eclipse, go to the Help menu, and select the Install New Software option. In the opening dialog, press the button Add, and write SeeIT 3D in the name field and http://davidmr.github.com/seeit3d/update in the location field. Then, press the OK button. Once these options are loaded and processed, select the SeeIT 3D Core checkbox and continue with the installation process. At this point you may receive a security warning related to unsigned content. Just press the OK button to continue. Finally, the installation process will ask you to restart the IDE. After you restart Eclipse, SeeIT 3D will be ready to work. System requeriments Eclipse 3.7
